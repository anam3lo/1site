Seletores em CSS

Um seletor é a primeira parte de uma regra CSS. Ele é responsável por selecionar o elemento que vai receber a estilização.

Propriedades CSS

Uma propriedade de um valor css é uma caracteristica especifica que será afetada pela regra, por exemplo, a cor do texto.

Valor CSS

O valor de uma propriedade é a grandeza ou aspecto visual que a propriedade recebe. Por exemplo, a espessura de uma borda em pixels ou mesmo a cor especifica para um fundo
